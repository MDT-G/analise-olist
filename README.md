Análise de comercio eletrônico Brasileiro — Olist

Dataset de um conjunto de dados públicos, foi analisado o mês com mais pedidos, o estado com mais pedidos e a categoria mais vendida.

Mês mais pedido = Agosto com 10.843 pedidos;
Estado mais pedido = SP com 41.746 pedidos;
Categoria mais pedida = cama_mesa_banho com 11.115 pedidos.

Nessa analise utilizei Python, Pandas e kagglehub.
